# Finance_Chatbot_AgenticAI_RAG_MultiModal_CrewAI
Gradio and Streamlit based Chatbot for Financial Queries using CrewAI Framework to showcase RAG based AgenticAI in python v3.11.0
- This Repo includes sample inputs/PDFs used, requirements.txt, screenshot of the chatbot, scripts
- Script uses groq LLM(llama) and huggingface embeddings
- Install packages as "python.exe -m pip install -r Py3p11p0_Finance_CrewAI_requirements.txt"
- Parsing the Input PDF using CrewAI Tools
- Modify the paths as required in the scripts
- Run chatbot script as "python.exe filename.py" for gradio and "python.exe -m streamlit run filename.py" for streamlit
